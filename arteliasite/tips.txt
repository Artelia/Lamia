 * pgadmin : 
 1 dbase : properties/security : select the user and "connect" (= GRANT CONNECT ON DATABASE database_name TO username;)
 2 schema : Properties/security : select the user and "usage" (=GRANT USAGE ON SCHEMA schema_name TO username;)
 3  schema : Properties/default privilege : select the user and "select" (=ALTER DEFAULT PRIVILEGES IN SCHEMA public    GRANT SELECT ON TABLES TO xxx;;)
  but this does not affect existing tables - so make the query:
  GRANT SELECT ON ALL TABLES IN SCHEMA public TO xxx;    for applying to existing tables
  
  NB : REVOKE SELECT ON ALL TABLES IN SCHEMA public FROM user;     to revoke
